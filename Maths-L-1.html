Les suites numériques

I. Notion de suites

1) Activité

Chercher les nombres manquants dans les listes suivantes :

  • L1 : 1 ; 3 ; 5 ; 7 ; 9 ; 11
  • L2 : 2 ; 4 ; 8 ; 16 ; 32 ; 64
  • L3 : 0 ; 1 ; 3 ; 5 ; 8 ; 12 ; 17
  • L4 : 1 ; 10 ; 100 ; 1000 ; 10000 ; 100000
  • L5 : 10 ; 13 ; 16 ; 19 ; 22 ; 25
  • L6 : 0 ; 1 ; 1 ; 2 ; 3 ; 5 ; 8 ; 13 ; 21

2) Définitions

Définition : Une suite (un) consiste à associer à tout nombre entier naturel n un nombre réel noté un. On dit que un est le terme d'indice n ou de rang n.

Si un est le terme général, un+1 est le terme suivant de un.

Exemple 1 :

Les nombres de la liste L1 : 1 ; 3 ; 5 ; 7 ; ... définissent la suite des nombres impairs. Le premier terme est 1.

On peut noter la suite (un) et ses termes successifs : u0 = 1 ; u1 = 3 ; u2 = 5 ; ...

Exemple 2 :

Les nombres de la liste L2 : 1 ; 10 ; 100 ; 1000 ; .. définissent la suite des puissances de 10.

On peut alors considérer la suite (vn) avec : v0 = 1 ; v1 = 10 ; v2 = 100 ; ...

3) Modes de génération d'une suite

Une suite est une liste de nombres, mais elle peut être définie à l'aide d'une formule :

a) Suite définie de façon explicite

On connaît directement le terme d'indice \(n\) en fonction de \(n\). On peut alors calculer le terme 4.

Exemple :

\(u_n = 3n + 1\)

Exemple 2 :

Soit \(v_n\) la suite définie sur \( \mathbb{N} \) par \( v_n = 2n + 1 \).

b) Suite définie par récurrence

On connaît le premier terme et la formule qui permet de calculer un terme quelconque à partir du précédent.

Exemple 1 :

Soit la suite \(u_n\) définie par \(u_0 = 1\) et \(u_{n+1} = 2u_n + 3\).

Exemple 2 :

Soit la suite \(v_n\) définie par \(v_0 = 5\) et \(v_{n+1} = v_n + 3n\).

II. Variations d'une suite

1) Monotonie d'une suite (variations)

Une suite \((u_n)\) est croissante lorsque pour tout \(n\), \(u_n \leq u_{n+1}\).

Une suite \((u_n)\) est décroissante lorsque pour tout \(n\), \(u_n \geq u_{n+1}\).

Une suite \((u_n)\) est monotone lorsqu'elle est croissante ou décroissante.

2) Méthodes pour étudier le sens de variation

On cherche le sens de variation de \((u_n)\) en comparant \(u_{n+1}\) et \(u_n\).

Méthode 1 : On étudie le signe de la différence \(u_{n+1} - u_n\).

Méthode 2 : On étudie le sens de variation de \(f\) sur \([0 ; +\infty[\).

Méthode 3 : On compare le quotient \(\frac{u_{n+1}}{u_n}\) avec 1.

Exemples

Exemple 1 :

Soit \((u_n)\) définie par \(u_n = 7n + 1\) pour \(n \in \mathbb{N}\).

Exemple 2 :

Soit \((u_n)\) définie par \(u_n = n^2 - 4n + 5\) pour \(n \in \mathbb{N}\).

Exemple 3 :

Soit \((u_n)\) définie par \(u_n = \frac{2}{n}\) pour \(n \in \mathbb{N}^*\).

Remarque : Soit \((u_n)\) définie par \(u_n = 2 \times (-1)^n\). On a \(u_0 = ...\), \(u_1 = ...\), \(u_2 = ...\), etc.

Attention : Lorsque la suite \((u_n)\) est définie par une relation de récurrence \(u_{n+1} = f(u_n)\), les variations de \(f\) et de \((u_n)\) ne sont pas nécessairement les mêmes.
